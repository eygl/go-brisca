# go-brisca
Multiplayer Brisca game written in Go.

Player Joins Site -> give them a uuid, that will be their player id.
'Give player ability to have name'

Create Lobby -> Create new lobby object -> move player to that Lobby


Statuses
- Open
- InProgress
- Completed

gameState
- starting
- player1 turn
- player2 turn
- player3 turn
- player4 turn
- finished

Lobby
- lobbyID
- createdDate
- player1
- player2
- player3
- player4
- status 
- gameState
